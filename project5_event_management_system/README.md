# Event Management System (EMS) Project Briefing Report

This project involves developing a comprehensive Project Briefing Report for an Event Management System (EMS) as part of the *31272 Project Management and the Professional* subject at the University of Technology Sydney.  
The report simulates a professional consulting engagement, where a project team delivers a board-level planning proposal for an IT system solution.

## üîç Key Features
- **Business Overview:** Defined project objectives, scope, success criteria, measurable organisational value (MOV), and assumptions.
- **Project Management Strategy:** Developed stakeholder register, communication plan, and risk register (15+ identified risks with mitigations).
- **SDLC Selection:** Evaluated Waterfall vs. Agile vs. Hybrid approaches and selected a Hybrid model for flexibility and structure.
- **Scheduling & Estimation:** Constructed a 120-task Work Breakdown Structure (WBS), Activity-on-Node dependency diagram, and Gantt chart showing project timeline and critical path.
- **Budget Validation:** Conducted top-down and bottom-up cost estimation to verify financial feasibility and establish contingency allocations.
- **Quality & Change Control:** Designed monitoring and QA frameworks ensuring data integrity and alignment with client deliverables.
